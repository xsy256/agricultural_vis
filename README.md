# 🌾 耕地·人口·农业产量

交互式可视化项目，探究耕地面积和人口数量对农业供给需求的影响。

## 在线访问
[https://你的用户名.github.io/agricultural_vis/](https://你的用户名.github.io/agricultural_vis/)

## 功能特点
- 散点气泡图展示全球国家农业数据
- 支持年份滑动、区域/收入筛选
- 可切换生产指数/营养不良率两种指标
- 点击国家查看详细信息

## 技术栈
- D3.js v7
- 原生 JavaScript
- HTML5/CSS3

## 数据来源
World Bank Open Data
