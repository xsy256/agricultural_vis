💡 设计理念

核心问题：在全球人口持续增长、气候变化加剧的背景下，哪些国家有能力养活自己的人民？耕地面积大的国家就不会面临粮食安全问题吗？人口大国必定面临很大的粮食安全压力吗？哪些国家依赖着其他国家维持粮食安全，又是哪些国家在“养活世界”？粮食安全与人口、耕地、经济发展之间存在怎样的关系？

本可视化工具旨在通过交互式探索，帮助用户直观理解全球农业和粮食安全的分布格局和影响机制。

🎯

1. 多维度透视

单一指标无法全面反映粮食安全状况。本工具整合了人口规模、耕地面积、谷物单产、生产供给、营养状况、贸易角色等8个维度，提供立体的分析视角。

🔄

2. 交互式探索

用户可以通过年份滑块观察时间变化，通过区域/收入筛选聚焦特定群体，通过点击国家深入查看详情。探索路径由用户自由决定。

📐

3. 视觉编码统一

散点图中：位置代表人口规模与核心指标，大小代表耕地产量，颜色代表区域或贸易角色。雷达图中：外圈始终表示表现更好。视觉语言一致，降低认知负担。

🗺️

4. 空间认知辅助

右侧真实地图辅助用户建立空间位置认知，将抽象的统计数据与具体的地理位置关联起来，增强理解深度。

指标设计逻辑

本工具中的所有指标均来自世界银行开放数据，经过以下处理以增强可读性：

原始指标单位可视化处理设计理由
人口人对数转换（log10）缩小数据范围，避免人口大国过度挤压空间
耕地产量万吨平方根映射为气泡半径平衡视觉权重，避免极端值气泡过大
营养不良率%反转后纳入雷达图保持"外圈=表现好"的统一规则
人均食品生产指数基准100乘以10后归一化与其他指标数值范围匹配

色彩设计

  • 大洲配色：采用色盲友好的 distinct 色板，确保各区域可区分
  • 贸易角色配色：
    • 🟢 绿色系：出口国（从浅绿到深绿表示出口强度）
    • 🟠 橙色系：进口国（从橙色到红色表示进口依赖程度）
    • ⚪ 灰色：平衡型或数据缺失
  • 高亮国家：选中时黑色边框+橙色填充，与其他气泡形成鲜明对比

交互反馈设计

  • 悬停提示：鼠标悬停气泡时显示国家名称和产量概要
  • 点击联动：点击气泡后右侧面板、雷达图、世界地图同步更新
  • 筛选联动：区域/收入筛选后，散点图动态更新，已选中国家自动清除
  • 年份过渡：滑块拖动时，气泡位置、大小、颜色平滑过渡（300ms 动画）

🎯 功能特点

📊 散点气泡图

  • X轴：人口数量（对数尺度）—— 反映国家人口规模
  • Y轴：可切换两种指标
    • 📊 农业生产供给力度：基于人均食品生产指数，反映国家农业产出能力
    • ⚠️ 营养不良率 (%)：反映粮食安全状况，数值越低越好
  • 气泡大小：耕地产量（耕地面积 × 谷物单产），越大表示农业产出越丰富
  • 气泡颜色：可切换两种模式
    • 🌍 大洲：按地理区域区分（亚洲/非洲/欧洲/北美洲/南美洲/大洋洲/加勒比）
    • 📦 贸易角色：按粮食进出口类型区分（强劲出口国/净出口国/净进口国/严重进口依赖/基本平衡）

控件功能

🌍 区域筛选按亚洲、非洲、欧洲、北美洲、南美洲、大洋洲、加勒比筛选国家
💰 收入组筛选高收入 / 中高等收入 / 中低等收入 / 低收入
🇺🇳 国家选择下拉菜单快速跳转至特定国家
🎨 颜色模式切换大洲 / 贸易角色
🔄 重置/清除按钮恢复筛选条件或清除选中国家

📋 国家详情面板

点击任意气泡后，右侧面板会显示该国的详细信息：

  • 🏷️ 国家标识：贸易角色徽章 + 粮食安全评级
  • 🔵 八维雷达图：多维度对比农业表现
  • 🗺️ 实时世界地图：基于真实地理数据，自动高亮选中国家（小岛国自动添加标记点）

📖 使用指南

第一步：浏览全局

打开页面后，你会看到一个散点气泡图，每个气泡代表一个国家。气泡位置由人口规模和所选指标决定，气泡大小代表耕地产量。

第二步：筛选数据

使用顶部的控制面板：

  • 拖动年份滑块，观察不同年份的数据变化
  • 选择区域或收入组，聚焦特定类型的国家
  • 切换颜色模式，对比不同分类下的国家分布

第三步：探索细节

点击任意气泡，右侧面板会展示该国详细数据：

  • 八维雷达图展示该国在全球范围内的相对表现
  • 真实世界地图高亮显示该国位置（小岛国自动添加标记点）
  • 贸易角色和粮食安全评级一目了然

第四步：切换分析视角

  • 点击 生产指数 / 营养不良率 按钮，切换 Y 轴指标
  • 点击 大洲 / 贸易角色 按钮，切换颜色模式

🔍 用户可以探索的有趣结论

📌 结论一：人口大国 ≠ 粮食安全

中国、印度等人口大国气泡巨大（人口×产量），但切换Y轴到"营养不良率"后，这些国家的位置可能并不理想，说明人口规模大并不等同于粮食安全状况好。相比之下，美国、巴西等农业出口国在供给力度指标上表现突出。

📌 结论二：贸易角色与地理分布强相关

切换颜色模式为贸易角色，可以清晰看到：

  • 🌾 强劲出口国：美国、巴西、阿根廷、乌克兰、法国等（多为美洲和欧洲国家）
  • 🛒 严重进口依赖：多数非洲国家、中东国家、日本、韩国等
  • ⚖️ 基本平衡：部分欧洲国家和亚洲国家

📌 结论三：营养不良率的地域差异

切换Y轴为营养不良率，非洲和南亚国家普遍位于图表上方（高营养不良率），欧洲和北美国家位于图表下方（低营养不良率）。值得注意的是，部分高收入国家（如日本、韩国）虽营养充足，但粮食自给率低，表现为"严重进口依赖"。

📌 结论四：耕地面积与产量的非线性关系

气泡大小代表耕地产量。观察发现：耕地面积大的国家（如俄罗斯、加拿大）气泡并不一定大——因为单位面积产量（单产）较低。而集约化农业国家（如荷兰、日本）耕地面积小但气泡不小，反映了高科技农业的高效率。

📌 结论五：时间趋势观察

滑动年份滑块：

  • 中国、印度等发展中国家的气泡随时间逐渐变大（产量提升）
  • 部分非洲国家的营养不良率随时间改善（气泡位置下移）
  • 战乱国家（如叙利亚、也门）各项指标出现恶化

📌 结论六：粮食安全与收入的关联

使用收入组筛选器，可以对比不同收入水平国家的粮食安全状况。高收入国家普遍营养不良率较低，但部分高收入国家（如石油输出国）高度依赖粮食进口。

📡 雷达图解读

右侧面板的八维雷达图从以下维度评估各国农业表现（外圈 = 表现更好）：

  • 耕地产量：农作物总产量（万吨）
  • 耕地面积：农业用地面积（万 km²）
  • 谷物单产：单位面积产量（kg/公顷），反映农业技术水平
  • 供给力度：基于人均食品生产指数，数值越大表示供给越充足
  • 粮食安全：基于营养不良率转换，值越小越靠内表示状况越好（外圈=安全）
  • 粮食出口占比：粮食出口占贸易总额比例（%）
  • 粮食进口占比：粮食进口占贸易总额比例（%）
  • 总人口：国家人口规模（亿人）

💡 提示：雷达图五边形越往外扩张，表示该国在该维度上表现越优秀。营养不良率经过反转处理，因此"粮食安全"轴外圈表示低营养不良率（安全）。
